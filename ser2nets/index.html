Serial to Network Proxy enhanced (ser2nets)

ser2net is a program for allowing network connections to serial ports. It provides all the serial port setup, a configuration file to configure the ports, a control login for modifying port parameters, monitoring ports, and controlling ports.

ser2nets is an enhanced version of ser2net with the following features:

  • multiple connections to the same serial port
  • multiple ports of a serial port
  • new "http" mode for WebSocket connections
  • new "rraw" mode for actively connecting to a remote port instead of listening locally
